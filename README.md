# Mo AI Studio

Mo AI Studio 是一个企业级 AI 智能体运行平台，能够运行各种定制的智能 AI 代理。它为这些 AI 代理提供了系统级的能力，使它们能够操作系统文件、编写和修改文档、回答问题以及共享知识，从而帮助企业实现真正的智能化协同。

## 特性

- 支持全种类 IDE 和编程语言
- 能一次修改大量文件，实现正确推理
- 支持跨项目的上下文修改
- 自定义智能体
- 系统级文件操作能力
- 文档编写和修改
- 问答功能
- 知识共享
